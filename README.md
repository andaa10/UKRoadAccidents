# UK Traffic Accidents Analysis using (Excel & SQL)

## Project Overview
This project analyzes traffic accident data in the United Kingdom to identify key factors affecting road safety. The focus is on understanding accident patterns, traffic behavior, and the role of environmental conditions using **Excel** and **SQL**.


## Objectives
* Analyze accident trends across different regions
* Identify key factors influencing accident frequency
* Explore temporal patterns (rush hours, weekdays vs weekends)
* Evaluate the impact of weather, lighting, and road conditions
* Build dashboards for data-driven insights


## Data Challenges
* Duplicate records
* Missing values
* Inconsistent formatting
* Unbalanced data distribution
These issues were handled through data cleaning and preprocessing in Excel and SQL.


## Tools Used
* *Excel*
  * Data cleaning and preprocessing
  * Pivot tables and data modeling
  * Interactive dashboards and visualizations
* *SQL*
  * Data querying and filtering
  * Aggregations and grouping
  * Extracting insights from structured data

## Files Included
#### Excel Files
* Cleaned dataset sheet
* Pivot tables and analysis sheets
* Dashboard sheets showing: Accident trends by time, Regional distribution, Severity levels, Vehicle involvement, etc..
#### SQL Files
* Queries for: Accident frequency analysis, Time-based trends (rush hours), Vehicle and casualty insights, Environmental factor evaluation, etc..


## Key Insights
* Traffic congestion is the **main factor** behind accidents
* Peak hours (7–9 AM, 3–5 PM) have the highest accident rates
* Urban areas show higher accident frequency than rural areas
* Weather, lighting, and road type have **minimal impact**
* Most accidents are **slight**, with fewer serious and fatal cases


## Recommendations
* Improve traffic management during peak hours
* Optimize urban traffic flow (signals, lanes, roundabouts)
* Increase awareness about rush-hour risks
* Enforce traffic rules in high-risk areas
* Use real-time traffic monitoring systems
